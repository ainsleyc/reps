
First in Coffeescript:
* Arrows
* Classes
* Template strings
* Destructuring
* Default
* For..Of

Other
* Let
* Const
* Modules
* Map, WeakMap
* Set, WeakSet
* Symbols
* Promises
* Iterators
* Generators
  * Special type of function, declared with "*function()"
  * Can "yield" to stop execution, sends out the yield value
  * Executing a generator function returns an iterator
  * Calling iterator.next() returns the yielded value
  * Passing iterator.next(x) passes in the x value to the next yield
